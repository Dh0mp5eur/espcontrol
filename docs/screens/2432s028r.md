# 2.8-inch 2432S028R

The **ESP32-2432S028R**, commonly known as the **Cheap Yellow Display (CYD)**, is a compact and affordable 2.8-inch touchscreen powered by an **ESP32** processor. With a 320×240 display it fits **6 cards** on the home screen — perfect as a small bedside or desk controller.

## Specifications

| | |
|---|---|
| **Screen size** | 2.8 inches |
| **Resolution** | 320 × 240 |
| **Orientation** | Landscape |
| **Display interface** | SPI (ILI9341) |
| **Processor** | ESP32 |
| **Flash** | 4 MB |
| **WiFi** | Built-in (2.4 GHz) |
| **Touch** | XPT2046 resistive |
| **Power** | Micro-USB |

## Card Grid

The home screen uses a **3-row × 2-column** grid, giving you **6 card slots**. Any home-screen card can be turned into a [Subpage](/features/subpages) folder containing up to 8 more cards.

Flexible card sizes are supported: Single, Tall, Wide, and Large.

## ESPHome Manual Setup

If you use ESPHome and prefer to compile firmware yourself:

```yaml
substitutions:
  name: "desk-panel"
  friendly_name: "Desk Panel"

wifi:
  ssid: !secret wifi_ssid
  password: !secret wifi_password

packages:
  setup:
    url: https://github.com/Dh0mp5eur/espcontrol/
    file: devices/esp32-2432s028r/packages.yaml
    refresh: 1sec
```
