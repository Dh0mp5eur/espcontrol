# 4.3-inch 8048S043C

The **Sunton ESP32-S3-8048S043C** is a 4.3-inch landscape touchscreen powered by an **ESP32-S3** processor. With an 800×480 display it fits **12 cards** on the home screen, making it ideal as a wall-mounted control panel.

## Specifications

| | |
|---|---|
| **Screen size** | 4.3 inches |
| **Resolution** | 800 × 480 |
| **Orientation** | Landscape |
| **Display interface** | DPI RGB |
| **Processor** | ESP32-S3 |
| **WiFi** | Built-in (2.4 GHz) |
| **PSRAM** | Octal, 80 MHz |
| **Touch** | GT911 capacitive |
| **Power** | USB-C |

## Card Grid

The home screen uses a **3-row × 4-column** grid, giving you **12 card slots**. Any home-screen card can be turned into a [Subpage](/features/subpages) folder containing up to 8 more cards.

Flexible card sizes are supported: Single, Tall, Wide, and Large.

For a full walkthrough including WiFi setup and Home Assistant pairing, see the [Install guide](/getting-started/install).

::: tip After flashing or OTA update
This panel uses DPI RGB with octal PSRAM, which requires a brief hardware reset after flashing or OTA updates. The firmware handles this automatically with a short deep-sleep cycle — the display may flicker once before coming back up normally.
:::

## ESPHome Manual Setup

If you use ESPHome and prefer to compile firmware yourself:

```yaml
substitutions:
  name: "wall-panel"
  friendly_name: "Wall Panel"

wifi:
  ssid: !secret wifi_ssid
  password: !secret wifi_password

packages:
  setup:
    url: https://github.com/Dh0mp5eur/espcontrol/
    file: devices/sunton-esp32-s3-8048s043c/packages.yaml
    refresh: 1sec
```
